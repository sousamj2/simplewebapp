Welcome to Mostly Jovial Crafters! Here's everything you need to know to get playing.

Connect to the Server

3

Start the Server

To save costs, the server is automatically suspended when no one is online. To bring it back up:

  1. Log in to this portal.
  2. Go to your Profile page.
  3. Click "Start Server" and confirm with the code sent to your email.
  4. Wait 30 seconds to 2 minutes for the server to come online.

⚠️ If the server doesn't respond within 5 minutes, the admin is automatically notified.

4

Your Starting Rank

When you first log in to the game, you'll be assigned the "No Rank" status. You can immediately be upgraded to Rank 1: Player for €1/month.

Rank 1 (Player) is the standard playing rank — the admin plays at this rank too, so it's a full experience with no "pay-to-win" disadvantage.

There is also a special perk: if you help keep the server running by following the startup procedure above, you can get a discount on your rank fee.
