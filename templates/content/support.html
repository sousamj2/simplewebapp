🖥️ How the Server Works

The server runs on Google Cloud Platform (GCP) and is paid for personally by the admin. To keep costs manageable, the server is automatically suspended when no players are online.

To start the server:

  1. Log in to mc.mjcrafts.pt (this website)
  2. Go to your Profile page
  3. Click "Start Server" and confirm with the code sent to your email
  4. Wait 30 seconds to 2 minutes — the server will come online automatically

If the server does not respond within 5 minutes, the admin is notified and will investigate as soon as possible.

💡 If a PATREON sponsor supports the server, it can run 24/7 (~€150/year), removing the need to start it manually.

💬 Contact & Reporting

For bug reports, rule violations, or general questions, please reach out via Discord. You can also report incidents involving harassment or discrimination directly to the admin in-game or via Discord.
